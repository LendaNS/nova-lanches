CYVIIH PAPELARIA - PROJETO PRONTO

Como usar:
1. Extraia o ZIP.
2. Abra a pasta no VS Code.
3. Rode no terminal:
   npm install
   npm run dev

Para publicar na Vercel:
1. Envie essa pasta para um repositório no GitHub.
2. Importe o repositório na Vercel.
3. A Vercel detecta Vite automaticamente.
